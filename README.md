# GitHub Safe Love Website

Website ini dibuat agar foto di halaman kedua tetap muncul di GitHub Pages.

PENTING:
- Foto sudah di-embed langsung di `result.html` memakai base64.
- Jadi tidak perlu upload folder `assets`.
- Upload file berikut ke root repository GitHub:
  - `index.html`
  - `result.html`
  - `style.css`
  - `script.js`

Setelah upload, aktifkan GitHub Pages dari Settings > Pages.
